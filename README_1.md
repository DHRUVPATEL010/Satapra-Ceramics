# Images Folder Structure

This folder contains all images organized by page and category for easy management.

## üìÅ Folder Structure

### Main Page Folders
- **`home/`** - Images for the home page (index.html)
  - Carousel images
  - Hero section images
  - Home page specific graphics

- **`contact/`** - Images for the contact page
  - Contact page specific images
  - Location images
  - Team photos

- **`about/`** - Images for the about us page
  - Company photos
  - Team images
  - Facility images
  - History photos

- **`products/`** - Images for the products page
  - Product catalog images
  - Product showcase images

- **`general/`** - General images used across multiple pages
  - Logos
  - Icons
  - Background images
  - Common graphics

### Product Category Folders
- **`categories/wall-tiles/`** - Wall tile product images
- **`categories/floor-tiles/`** - Floor tile product images
- **`categories/bathroom-tiles/`** - Bathroom tile product images
- **`categories/kitchen-tiles/`** - Kitchen tile product images
- **`categories/parking-tiles/`** - Parking tile product images
- **`categories/sanitary-ware/`** - Sanitary ware product images
- **`categories/vitrified-tiles/`** - Vitrified tile product images
- **`categories/ceramic-tiles/`** - Ceramic tile product images
- **`categories/porcelain-tiles/`** - Porcelain tile product images
- **`categories/marble-tiles/`** - Marble tile product images

## üìù Image Naming Convention

### Home Page Images
- `carousel-1.jpg` - First carousel slide
- `carousel-2.jpg` - Second carousel slide
- `hero-banner.jpg` - Main hero image
- `feature-1.jpg` - Feature image 1

### Product Category Images
- `category-banner.jpg` - Category page banner
- `product-1.jpg` - Product image 1
- `product-2.jpg` - Product image 2
- `gallery-1.jpg` - Gallery image 1

### General Images
- `logo.png` - Company logo
- `favicon.ico` - Website favicon
- `background-pattern.jpg` - Background pattern

## üéØ Usage Examples

### In HTML files:
```html
<!-- Home page carousel -->
<img src="images/home/carousel-1.jpg" alt="Wall Tiles">

<!-- Product category -->
<img src="images/categories/wall-tiles/product-1.jpg" alt="Wall Tile Product">

<!-- General logo -->
<img src="images/general/logo.png" alt="Company Logo">
```

## üìã To Add New Images:

1. **Identify the page** where the image will be used
2. **Choose the appropriate folder** from the structure above
3. **Name the image** following the naming convention
4. **Update the HTML file** to reference the new image path
5. **Test the image** displays correctly on the website
